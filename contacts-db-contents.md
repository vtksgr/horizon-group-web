# Contact

### **Company Contact（企業様へお問い合わせ）**
required: company name (会社名)
required: First name (姓) 
required: Last name（名）
required: Managers/Executives (**経営者/役員**)
Ratio button:
    1. Manager （部長）
    2. Section Chief/Manager（課長／マネージャー）
    3. Chief/Section Chief　（主任/係⻑）
    4. General employees　（⼀般社員）
required: email （メール）
required: Phone number （電話番号）
required: Postal Code （郵便番号）
required: Company Address （**会社所在地 ）**
required: Inquiry type （**お問い合わせ種類）**
Radio Button：
    1. Recruitment　（人材紹介）
    2. International Student Support　（留学生支援）
    3. Interpretation/Translation　（通訳・翻訳）
    4. Specified Skilled Worker　（特定技能）
    5. Registration Support　（登録⽀援）
    6. Education/Consulting　（教育・コンサルティング）
    7. Other Inquiries （その他のお問い合わせ）(If the other  option is checked, make the Message  field required.)
required:  Message (お問い合わせ内容)
